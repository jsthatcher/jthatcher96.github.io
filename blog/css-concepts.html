What is the difference between margin, border, and padding?

All three of these can be used to space your website creatively. The margin of an object refers to the space surrounding it. Margins don't have a color or style, they're just empty space. A border immidiately surrounds the object. It can be seen and may have different styles, colors, and thickness. Padding separates the inside of the object from whatever else is in there (like a text box). It changes how tightly the outside object surrounds the inside object.
